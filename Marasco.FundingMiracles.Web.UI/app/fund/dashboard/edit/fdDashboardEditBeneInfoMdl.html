Adding a Beneficiary? Please read below

Just a couple of things to note before you add a beneficiary to this fund
  • The Beneficiary will need to setup their own account(s). [WePay, Miracles]. Don't worry, we'll assist them!
  • Only the beneficiary is allowed to withdraw from account.
  • You will not be required to setup a WePay account for this fund
  • Once set, you will not be able to change the beneficiary info.
